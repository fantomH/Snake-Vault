# Snake-Enigma

Hash, cipher, encryption...

Provides the command `snake-enigma`, which is the equivalent of `python -m snake_vault.snake_enigma`.

## md5

MD5 manipulation.

### Command `snake-enigma [command] [options]`

```shell
# --- Calculate the md5sum of a file
snake-enigma file-md5sum file
```

#### file_md5sum(filepath)

* filepath: str. input file.
